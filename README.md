# Website Saimon Daminelli - Analista de TI e BI

Este é um website profissional completo desenvolvido com HTML5, CSS3 e JavaScript, apresentando o portfólio e serviços de Saimon Daminelli, especialista em Business Intelligence e automação de dados.

## 🚀 Funcionalidades Implementadas

### ✅ Estrutura Completa
- **3 páginas HTML**: Home, Sobre e Contato
- **Cabeçalho e rodapé compartilhados** em todas as páginas
- **Navegação funcional** entre páginas com menu dropdown
- **Estrutura semântica**: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`

### ✅ SEO e Acessibilidade
- **Meta descrições** personalizadas para cada página
- **Favicon personalizado** com as iniciais "SD"
- **Títulos únicos** para cada página
- **Design responsivo** para desktop e mobile

### ✅ Sistema de Navegação
- **Menu principal** com links funcionais
- **Menu dropdown** para serviços
- **Pseudoclasses CSS** para efeitos hover, active e focus
- **Indicação visual** da página ativa

### ✅ Formulário de Contato Completo
- **Nome** (campo obrigatório)
- **E-mail** (campo obrigatório)
- **Senha** (campo obrigatório)
- **Gênero** (radio buttons com 4 opções)
- **Área de interesse** (select com 6 opções)
- **Faixa de orçamento** (select opcional)
- **Mensagem/comentários** (textarea obrigatório)
- **Checkboxes** para newsletter e termos
- **Validação JavaScript** com animações

### ✅ Efeitos Visuais Avançados
- **Transições suaves** em todos os elementos interativos
- **Animações CSS** (float, pulse, glow, fade, slide)
- **Box-shadow e text-shadow** para profundidade
- **Transformações** (scale, rotate, translate)
- **Gradientes** e efeitos de hover
- **Partículas animadas** no background
- **Scroll indicator** animado
- **Efeitos de loading** nos formulários

### ✅ Interatividade JavaScript
- **Smooth scrolling** para âncoras
- **Intersection Observer** para animações de scroll
- **Validação de formulário** com feedback visual
- **Efeitos de hover** dinâmicos
- **Dropdown funcional** com JavaScript
- **Animações escalonadas** para listas
